# Mechanical Engineering Portfolio Website

This is a static portfolio site for mechanical engineering projects. It can be hosted anywhere that serves HTML, CSS, and JavaScript.

## Files

- `index.html` contains the page structure.
- `styles.css` contains the visual design and responsive layout.
- `script.js` contains the project list, filters, and selected-project details.

## Update Your Details

Edit these details before publishing if they change:

- Contact links in `index.html`
- The `projects` array in `script.js`
- Project summaries, tools, and outcomes in `script.js`

## Fastest Deploy Options

### Netlify Drop

1. Go to `https://app.netlify.com/drop`.
2. Drag this whole folder into the drop area.
3. Netlify gives you a public URL.

### GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, `.nojekyll`, and this `README.md`.
3. Open the repository `Settings`.
4. Go to `Pages`.
5. Set the source to deploy from the `main` branch and the repository root.
6. Save. GitHub will publish the site at a `github.io` URL.

### Custom Domain

After the site is live, connect a domain through your host's domain settings. Common domain registrars include Namecheap, Cloudflare, and Google Domains alternatives.
